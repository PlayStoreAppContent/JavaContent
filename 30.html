Java - throw & throws Keyword

 

 

throws keyword :-
The main purpose of 'throws' keyword is to delegate the resposibilities of exception handling from present method to the caller method. It requires in the case of checked exception.

 

import java.io.*;

 

 //Start of ThrowsDemo class

class ThrowsDemo

{

//Start of Main Method

  public static void main(String args[])

  {

displayMethod();

    }//End of Main Method

//Start of displayMethod()

static void displayMethod()  //Error

 {

   throwsExc();

   

   }//End of displayMethod

   

   //Start of throwsExc() Method

static void throwsExc() throws Exception

{

 BufferReader br = new BufferReader(new InputStreamReader(System.in));

 System.out.println("Enter your Name:");

 String name = br.readLine();

 System.out.println("Your Name is:" +name);

 

  }//End of throwsExc() Method

 

}//End of ThrowsDemo


If we are compiling the above code then we are getting the Compile time Error like:
ThrowsDemo.java: unreported exception java.lang.Exception; must be caught or declared to be thrown
throwsExc();

In the example-2 program to bypassing the generated exception throws from present method to caller method by using throws keyword.

 

 

import java.io.*;

 

 //Start of ThrowsDemo class

class ThrowsDemo

{

//Start of Main Method

  public static void main(String args[])

  {

displayMethod();

    }//End of Main Method

//Start of displayMethod()

static void displayMethod() throws Exception

 {

   throwsExc();

   

   }//End of displayMethod

   

   //Start of throwsExc() Method

static void throwsExc() throws Exception

{

 BufferReader br = new BufferReader(new InputStreamReader(System.in));

 System.out.println("Enter your Name:");

 String name = br.readLine();

 System.out.println("Your Name is:" +name);

 

  }//End of throwsExc() Method

 

}//End of ThrowsDemo

 

Save this file as ThrowsDemo.java

 

To compile: javac ThrowsDemo.java

To execute: java ThrowsDemo

Output:-

 

Enter Your Name:

Tarun

Your Name is: Tarun


throw keyword :-

Sometimes we can create Exception object explicitly and we can handover that exception object to the JVM explicitly by throw keyword. The purpose of throw keyword is to handover our created exception object explicitly to the JVM.

 

//Start of ThrowDemo class

class ThrowDemo

 {

  static void demoprog()

  {

    try {

      throw new NullPointerException("ThrowDemo");

 

    } catch (NullPointerException e){

      System.out.println("Caught inside demoprog.");

 

      throw e; // rethrow the exception

    }

  }

//Start of Main Method

  public static void main(String args[]) {

    try {

      demoprog();

    } catch (NullPointerException e) {

      System.out.println("Recaught: " + e);

    }

  }//End of Main Method

 

}//End of ThrowDemo class

 

Save this file as ThrowDemo.java

 

To compile: javac ThrowDemo.java

To execute: java ThrowDemo

Output:-

Caught inside demoprog.

Recaught:java.lang.NullPointerException:ThrowDemo
